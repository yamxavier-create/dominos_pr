  <action>
1. Create `railway.toml` at repo root with this exact content:
```toml
[build]
buildCommand = "npm run build"

[deploy]
startCommand = "npm start"
healthcheckPath = "/health"
healthcheckTimeout = 300
restartPolicyType = "ON_FAILURE"
restartPolicyMaxRetries = 5
```

2. In `server/src/index.ts`, add the health endpoint BEFORE the production static-serving block (the `if (config.NODE_ENV === 'production')` block). This is critical — if placed after, the `app.get('*', ...)` catch-all swallows it:
```typescript
// Health check for Railway
app.get('/health', (_req, res) => {
  res.status(200).json({ status: 'ok' })
})
```

3. Make CORS conditional — development only. Replace the current unconditional `app.use(cors(...))` with:
```typescript
if (config.NODE_ENV !== 'production') {
  app.use(cors({ origin: config.CLIENT_ORIGIN }))
}
```

4. Make Socket.io CORS conditional. Replace the current `cors` option in `new Server(httpServer, {...})` with:
```typescript
const io = new Server(httpServer, {
  cors: config.NODE_ENV !== 'production'
    ? { origin: config.CLIENT_ORIGIN, methods: ['GET', 'POST'] }
    : undefined,
  pingTimeout: 60000,
  pingInterval: 25000,
})
```

Keep `express.json()` middleware unchanged (it is not CORS-related).

Important ordering in index.ts (top to bottom):
  a. Conditional CORS middleware
  b. express.json()
  c. Health endpoint
  d. Production static serving block (with catch-all)
  e. Socket.io connection handler
  f. httpServer.listen
