1. Generate a lo-fi background music loop using ffmpeg (30 seconds, 96kbps, ~360KB):
   ```bash
   ffmpeg -f lavfi -i "sine=frequency=220:duration=30" \
     -af "atempo=0.8,aecho=0.8:0.88:60:0.4,lowpass=f=800,volume=0.4" \
     -b:a 96k client/public/audio/lofi-loop.mp3
   ```
   This is a functional placeholder -- trivially replaceable with a real lo-fi track later.

2. Create `client/src/audio/music.ts` -- a singleton module (NOT a React component) managing an HTML5 `<audio>` element:
   - Lazy-create a single `HTMLAudioElement` pointing to `/audio/lofi-loop.mp3`
   - Set `loop = true` and `volume = 0.3`
   - Export `playMusic()`: only calls `el.play()` if `el.paused` (prevents restarts on route changes within music routes). Catch rejected promise silently (autoplay policy).
   - Export `stopMusic()`: calls `audio.pause()` + resets `audio.currentTime = 0` (full stop for leaving music routes).
   - Export `pauseMusic()`: calls `audio.pause()` WITHOUT resetting currentTime (for toggle off, so toggling back on resumes).
   - Do NOT use AudioBuffer/Web Audio API -- HTML5 `<audio>` streams and has native loop support (see RESEARCH.md anti-patterns).

3. Create `client/src/hooks/useBackgroundMusic.ts`:
   - Import `useLocation` from react-router-dom, `useUIStore` from store, `playMusic`/`stopMusic`/`pauseMusic` from music.ts.
   - Define `MUSIC_ROUTES = ['/', '/lobby']`.
   - Single `useEffect` depending on `[pathname, musicEnabled]`:
     - If `MUSIC_ROUTES.includes(pathname) && musicEnabled` -> call `playMusic()` (no-op if already playing).
     - Else if `!musicEnabled` -> call `pauseMusic()` (preserves position for resume).
     - Else (route is `/game` or unknown) -> call `stopMusic()` (full reset).
   - No cleanup function needed -- singleton persists, lifecycle driven by route + toggle state.

4. Wire `useBackgroundMusic()` into `client/src/App.tsx`:
   - Add import: `import { useBackgroundMusic } from './hooks/useBackgroundMusic'`
   - Add `useBackgroundMusic()` call inside `AppRoutes()` right after `useSocket()`.
