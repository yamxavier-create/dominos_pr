1. Generate three MP3 audio files using ffmpeg (ffmpeg is available at /usr/local/bin/ffmpeg):

   Tile clack (short white noise burst, 0.1s):
   ```
   ffmpeg -f lavfi -i "anoisesrc=d=0.1:c=white:r=44100:a=0.5" -af "afade=t=out:st=0.05:d=0.05" -b:a 64k client/public/audio/tile-clack.mp3
   ```

   Turn notification (two-tone ascending chime, 0.3s):
   ```
   ffmpeg -f lavfi -i "sine=frequency=880:duration=0.15" -f lavfi -i "sine=frequency=1320:duration=0.15" -filter_complex "[0][1]concat=n=2:v=0:a=1,afade=t=out:st=0.2:d=0.1" -b:a 64k client/public/audio/turn-notify.mp3
   ```

   Pass sound (low thud tone, 0.15s):
   ```
   ffmpeg -f lavfi -i "sine=frequency=220:duration=0.15" -af "afade=t=in:d=0.02,afade=t=out:st=0.08:d=0.07" -b:a 64k client/public/audio/pass.mp3
   ```

2. Create `client/src/audio/sfx.ts` with:
   - `SFX_URLS` const object mapping `tileClack`, `turnNotify`, `pass` to their `/audio/*.mp3` paths
   - `buffers` partial record to hold preloaded AudioBuffers
   - `preloadSfx()` async function that calls `loadAudio()` for each URL, storing results in `buffers`
   - `playSfx(name, volume?)` function that checks `useUIStore.getState().sfxEnabled`, then calls `playBuffer()` if buffer exists
   - `playPassSfx()` function with leading-edge debounce (300ms window): plays the first pass sound immediately via `playSfx('pass')`, suppresses subsequent calls within 300ms. Uses a module-level `passDebounceTimer` variable.
   - Import `loadAudio` and `playBuffer` from `./audioLoader`, import `useUIStore` from `../store/uiStore`

Key details:
- `playSfx` accesses store via `useUIStore.getState()` (not a hook -- this runs outside React components)
- `preloadSfx` is fire-and-forget (non-blocking), called at game start
- Leading-edge debounce means: if timer is active, return early; otherwise play immediately and set timer
