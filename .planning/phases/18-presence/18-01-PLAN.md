**Class: PresenceManager**

Constructor takes `(io: Server, rooms: RoomManager)`. Stores them as private fields.

**Private state:**
- `connections: Map<string, Set<string>>` -- userId -> Set of socketIds
- `graceTimers: Map<string, NodeJS.Timeout>` -- userId -> pending offline timer

**Public methods:**

1. `addSocket(userId: string, socketId: string): void`
   - Clear any existing grace timer for this userId (via `this.graceTimers.get(userId)` -> `clearTimeout` -> `delete`)
   - Get or create the Set for this userId in `this.connections`
   - Check `wasOffline = !sockets || sockets.size === 0` BEFORE adding
   - Add socketId to the set
   - If `wasOffline`, call `this.broadcastStatusToFriends(userId)`

2. `removeSocket(userId: string, socketId: string): void`
   - Get the set from `this.connections`, return early if not found
   - Delete socketId from set
   - If `sockets.size === 0`, start grace period:
     - `setTimeout(() => { this.graceTimers.delete(userId); this.connections.delete(userId); this.broadcastStatusToFriends(userId) }, 5000)`
     - Store timer in `this.graceTimers.set(userId, timer)`

3. `getStatus(userId: string): PresenceStatus`
   - Get set from connections. If empty or missing, return `'offline'`
   - Call `this.rooms.getRoomCodeByUserId(userId)` -- if returns a roomCode:
     - Call `this.rooms.getRoom(roomCode)` -- if room exists:
       - Return `room.status === 'in_game' ? 'in_game' : 'in_lobby'`
   - Otherwise return `'online'`

4. `getStatuses(userIds: string[]): Record<string, PresenceStatus>`
   - Loop over userIds, call `this.getStatus()` for each, return object

5. `notifyStatusChange(userId: string): void`
   - Public method called after room events (create, join, leave, game start, game end)
   - Simply calls `this.broadcastStatusToFriends(userId)`

**Private methods:**

6. `async broadcastStatusToFriends(userId: string): Promise<void>`
   - Compute `status = this.getStatus(userId)`
   - Get friend IDs via `this.getFriendIds(userId)`
   - For each friendId, emit to `user:${friendId}` room:
     - Event: `'presence:friend_status_changed'`
     - Payload: `{ userId, status }`
   - Additionally, if status is `'online'` (just came online), also emit `'presence:friend_online'` with `{ userId, displayName }` -- query display name from the user lookup
   - If status is `'in_lobby'`, also emit `'presence:friend_in_lobby'` with `{ userId, displayName }`

7. `async getFriendIds(userId: string): Promise<string[]>`
   - Query Prisma: `prisma.friendship.findMany({ where: { status: 'ACCEPTED', OR: [{ requesterId: userId }, { targetId: userId }] }, select: { requesterId: true, targetId: true } })`
   - Map to return the OTHER user's ID (not userId)

For the `displayName` lookup in `broadcastStatusToFriends`, query: `prisma.user.findUnique({ where: { id: userId }, select: { displayName: true } })`. Only query when status is `'online'` or `'in_lobby'` (the two toast-worthy transitions).

Import prisma from `'../db/prisma'`, Server from `'socket.io'`, RoomManager from `'../game/RoomManager'`.
